# Git Basics

## Git CLI & Remote

- `git init`
- `git status`
- `git add <filename>`/ `git add .`
- `git commit -m "commit message"`
- `git push -u origin main` / `git push`

- `git remote -v`
- `git remote add origin <ssh link>`

## Git Branches & Prs

- `git switch -c <branch-name>`
- `git switch <branch-name>`
